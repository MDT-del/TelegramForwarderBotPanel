# پنل مدیریت ربات تلگرام فورواردر

یک ربات تلگرام برای فوروارد کردن پیام‌ها با قابلیت زمانبندی و پنل مدیریت وب.

## ویژگی‌ها

- 📤 **فوروارد پیام**: فوروارد متن، عکس، وویس، ویدیو و فایل‌های صوتی
- ⏰ **زمانبندی**: زمانبندی پست‌ها برای ارسال در آینده با استفاده از تقویم شمسی
- 🎛️ **پنل وب**: مدیریت تنظیمات ربات و مشاهده آمار از طریق رابط وب
- 📊 **آمارگیری**: پیگیری انواع پیام‌ها و آمار زمانبندی
- 🔐 **احراز هویت**: پنل ادمین امن با سیستم ورود
- 🐳 **پشتیبانی Docker**: استقرار آسان با Docker و Docker Compose

## پیش‌نیازها

- Python 3.9+
- Docker (اختیاری)
- توکن ربات تلگرام
- شناسه کاربر منبع
- شناسه چت مقصد

## نصب و راه‌اندازی

### استفاده از Docker (توصیه شده)

1. کلون کردن مخزن:
```bash
git clone <repository-url>
cd TelegramForwarderBotPanel
```

2. ایجاد فایل `.env`:
```bash
TELEGRAM_BOT_TOKEN=توکن_ربات_شما_اینجا
SOURCE_USER_ID=شناسه_کاربر_شما_اینجا
DESTINATION_CHAT_ID=شناسه_چت_مقصد_اینجا
SECRET_KEY=کلید_رمز_شما_اینجا
ADMIN_USERNAME=نام_کاربری_ادمین_شما
ADMIN_PASSWORD=رمز_عبور_امن_شما
```

3. اجرا با Docker Compose:
```bash
docker-compose up -d
```

### نصب دستی

1. نصب وابستگی‌ها:
```bash
pip install -r requirements.txt
```

2. تنظیم متغیرهای محیطی:
```bash
export TELEGRAM_BOT_TOKEN="توکن_ربات_شما_اینجا"
export SOURCE_USER_ID="شناسه_کاربر_شما_اینجا"
export DESTINATION_CHAT_ID="شناسه_چت_مقصد_اینجا"
export SECRET_KEY="کلید_رمز_شما_اینجا"
export ADMIN_USERNAME="نام_کاربری_ادمین_شما"
export ADMIN_PASSWORD="رمز_عبور_امن_شما"
```

3. اجرای برنامه:
```bash
python main.py
```

## نحوه استفاده

1. **پنل وب**: دسترسی به پنل مدیریت در `http://localhost:8080`
2. **دستورات ربات**: ارسال پیام به ربات برای فوروارد کردن
3. **زمانبندی**: استفاده از زمانبندی تعاملی برای تنظیم زمان‌های ارسال آینده

## تنظیمات

### متغیرهای محیطی

| متغیر | توضیحات | اجباری |
|-------|---------|--------|
| `TELEGRAM_BOT_TOKEN` | توکن ربات تلگرام شما | بله |
| `SOURCE_USER_ID` | شناسه کاربری که می‌تواند پیام ارسال کند | بله |
| `DESTINATION_CHAT_ID` | شناسه چت یا کانال مقصد | بله |
| `SECRET_KEY` | کلید رمز Flask برای نشست‌ها | خیر (خودکار تولید می‌شود) |
| `ADMIN_USERNAME` | نام کاربری پنل ادمین | بله |
| `ADMIN_PASSWORD` | رمز عبور پنل ادمین | بله |
| `MAX_LOGIN_ATTEMPTS` | حداکثر تلاش‌های ورود (پیش‌فرض: 3) | خیر |
| `SESSION_TIMEOUT` | زمان انقضای نشست به ثانیه (پیش‌فرض: 3600) | خیر |
